# Mob Egg Capture

This server-side only Fabric mod adds a Safari Net item that you can use to
capture friendly and hostile mobs in an item form to be released later.

Recipe for Safari Net:

![Recipe for Safari Net](media/safari_net_recipe.png)

Any egg can be used (normal, brown or blue)

Capturing hostile mobs requires them to have Weakness status effect. Bosses cannot be captured.

## Installation

This mod requires Fabric API to be installed. Mod does not need to be installed on the client.
